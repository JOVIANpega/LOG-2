此工具用於分析測試 LOG，並輸出 PASS/FAIL 匯總 Excel（僅針對 CENTIMANIA 格式）。以下只介紹最常用的按鈕與各標籤頁如何操作。

一、左側按鈕區

  • 📁 選擇單一檔案：挑一個 .log 檔進行分析。完成後可在三個標籤頁（PASS / FAIL / 原始LOG）查看結果。
  • 📂 選擇資料夾：一次分析資料夾內所有 .log 檔，並自動在該資料夾下建立 LOG集總整理/，輸出 PASS匯總.xlsx 與 FAIL匯總.xlsx。完成時跳出白底提示窗，可一鍵開啟輸出資料夾。
  • 🗑️ 清除結果：清空 PASS/FAIL/原始LOG 的畫面內容。
  • 📖 查看操作說明(HTML)：開啟本頁。

左上「已選擇：檔案/資料夾」會顯示目前的選擇，字體大小跟「內容字體」聯動。

二、PASS 標籤頁（✅ PASS測項）

  • 顯示所有通過的測試步驟清單，欄位：Step Name / 指令 / 回應 / 結果。
  • 內容字體可由左側「介面設定 → 內容字體」加減調整。
  • 若需檢視完整原始內容，切到「📄 原始LOG」。

三、FAIL 標籤頁（❌ FAIL測項）

  • 畫面上下分割：
    • 上方表格：欄位為 Step Name / 指令 / 錯誤回應 / Retry次數 / 錯誤原因。
    • 下方區域：顯示詳細錯誤內容，含紅色標示的 is Fail 文字。
  • 點選上方任一列，會即時更新下方詳細內容。
  • 打開檔案時會自動選取第一筆 FAIL 方便查看。

四、📄 原始LOG 標籤頁

  • 顯示完整原始 LOG 文字。
  • 若有錯誤行，會自動捲動到錯誤處並套用醒目標示。

五、匯出 Excel（僅「選擇資料夾」時自動執行）

  • 輸出路徑：所選資料夾內的 LOG集總整理/。
  • PASS匯總.xlsx：
    • Summary 工作表：
      • 欄位：檔名、PASS項目（超過 25 筆會水平分欄為 PASS項目2、3…）。
      • 相同 PASS 項目集合會合併成一組，檔名欄以換行列出所有對應檔名，包含 SFIS 與測試總秒數。
      • 底部提供「工作表快速連結（點擊跳轉）」；僅此區有超連結與懸停預覽（約 400x500）。
    • 各 LOG 工作表：第 1 行檔名（綠底白字），第 2 行 PASS 項目清單，第 3 行起貼上原始 LOG。
  • FAIL匯總.xlsx：
    • Summary 工作表：欄位 檔名 / FAIL原因，原因取自單檔主錯（例如：IPIO009-0580:Check Battery_tempture is Fail !）。
    • 各 LOG 工作表：第 1 行檔名（紅底白字），第 2 行主要錯誤原因，第 3 行 PASS 項目，第 4 行起為原始 LOG。

六、常見問題

  • 檔名是中文？→ 支援繁體中文檔名。
  • Excel 提示修復？→ 已做文字清理與格式保護；仍發生時，請先更新 Office 版本，並回報樣本 LOG。
  • 看不到預覽視窗？→ 只有 Summary 底部的「工作表快速連結」具備懸停預覽。
